Project Overview
Create a comprehensive, industry-grade frontend for a modern surveillance system with real-time human detection, geofencing, and alert management capabilities. The application should feature a stunning landing page, secure authentication, and a professional dashboard for monitoring multiple CCTV feeds with interactive maps.

Tech Stack Requirements
Frontend Framework: React , vite

Styling: Tailwind CSS 

State Management: Redux Toolkit or Zustand

Routing: React Router DOM

Maps: Leaflet/React-Leaflet for geospatial visualization

Charts: Recharts for analytics

Forms: React Hook Form with Zod validation

HTTP Client: Axios for API communication

WebSocket: Native WebSocket API for real-time updates

UI Components: Headless UI or Radix UI for accessibility

Icons: Lucide React or Heroicons

Notifications: React Hot Toast for user feedback
